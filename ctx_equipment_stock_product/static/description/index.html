Equipment Stock Product

Overview

This module help you to create equipment from stock products either from inventory adjustments or from purchase orders shipments.

Features
By installing this module, Inventory Lot/Serial number feature will be enabled automatically (if it's not already enabled.).
Ability to mark a product as equipment, and fill the default fields to be used in creating equipment.
Product tracking per serial number will enabled for each product marked as equipment.
Equipment will be created automatically either from inventory adjustments or from purchase orders received shipments.
Ability to view all Equipments linked to the product.
Ability to track the Equipment movement history.
Ability to track the current stock location for equipment.
Ability to view equipment linked to inventory serial number
How it works

Product Configuration

Once you mark the product as equipment, tracking per serial number will be set automatically.
