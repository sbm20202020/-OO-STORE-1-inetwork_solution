OVERVIEW

Now add Percent Field in your Odoo web without any extra effort.

FEATURES

  • Add Percent Field within a few seconds and ease your Odoo development process.
  • You can also use the Percent Field as a widget in XML to display the percent sign.
  • Use Percent Field as default Odoo field.
  • Validation is provided to ensure that the value of the Percent Field can only be numeric and between 0 to 100.
  • Note: Please add percent_field into depends of your manifest.
1

HOW IT WORKS

  • If you’re adding the Percent Field to an old, then do the same by writing widget = ”Percent” in XML wherever you wish to add the field.
2
  • If you’re adding the Percent Field to a new code, already working code, then do the same by writing fields.Percent() in python model wherever you wish to add the field.
