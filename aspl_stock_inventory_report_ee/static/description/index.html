Odoo 13 Stock Inventory Report (Enterprise)

Note: This module is tested and working in Odoo Enterprise edition. And we are supporting Ubuntu OS and not Windows OS.


Features

  • Generate Stock Inventory Report for Particular Date Range.
  • Generate Report in PDF and XLS Format.
  • Filter By Product and Category Wise.
  • Show Today Movement Product List in Report.
  • Send Email To Allowed Users Everyday.
  • Show Zero Quantity Product as well in report.
  • Group by Category wise show product in Report.

Available for Community

How to use

  • Select warehouses and Location for particular date range and generate PDF or XLS report.


  • show the product and it's inventory movement in PDF report for select the value in above configuration.


  • show the product and it's inventory movement in XLS report for select the value in above configuration.


  • PDF Report Group by Categories.


  • XLS Report Group by Categories.


  • Enable Today Movement in above configuration and print PDF report with and without Group By Categories.




  • Enable Today Movement in above configuration and print XLS report with and without Group By Categories.




  • as per below screenshot select user and email template. so everyday selected users will get email with only movement products in PDF report.
